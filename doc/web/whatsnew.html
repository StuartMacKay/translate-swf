What's New In Translate AS1

Here is a brief summary of all the releases of the Translate AS1 Compiler for ActionScript 1.x. The release notes link describe all the features, update and bug fixes included in each release.

Date Version Description Details
30th June 2006 2.0.3 Big improvements to the grammar with support for escaped characters in string, expressions using literal values and a number of bug fixes.. release notes
3rd April 2006 2.0.2 Major round of bug fixes and general improvements to the code generated by the compiler. release notes
14th June 2004 2.0.1 Translate was completely rewritten to boost performance of the parser and to allow encoded byte-codes and actions to be generated - allowing the framework to be integrated with any Java-based SDK used to generate Flash files. release notes
