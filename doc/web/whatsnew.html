What's New In Translate AS1

Here is a brief summary of all the releases of the Translate AS1 Compiler for ActionScript 1.x. The release notes link describe all the features, update and bug fixes included in each release.

Date Version Description Details
3rd April 2006 2.0.2 Major round of bug fixes and general improvements to the code generated by the compiler. release notes
14th June 2004 2.0 Translate was completely rewritten to boost performance of the parser and to allow encoded byte-codes and actions to be generated - allowing the framework to be integrated with any Java-based SDK used to generate Flash files. release notes
1.1.3 Update to use the revised API for classes in the Transform SWF framework. release notes
1.1.2 Bug fixes when using the 'new' keyword and loading web pages or movies. release notes
1.1.1 Maintenance release fixing a serious bug in array subscripting and added support for run-time evaluation of array, function and object names. release notes
1.1 The grammar used to parse scripts has been completely overhauled and now supports all of the language constructs and functions provided in Macromedia's Flash 5 ActionScript language. release notes
1.0 The first public release of the framework. release notes
